Vue는 User Interface(UI)를 빠르게 만들기 위한 자바스크립트 기반 프레임워크 이다. Vue를 사용하면 웹을 쉽고 빠르게 효율적으로 개발할 수 있다.  
  
Vue 애플리케이션을 만드는 방법은 아래와 같다.  
  
최신 버전의 노드가 설치되어있는지 확인 후  
```shell  
npm init vue@latest  
```  
  
명령어를 입력하면 `create-vue`를 설치 및 실행하며, TypeScript 및 테스트 지원과 같은 몇 가지 선택적 기능에 대한 프롬포트가 표시된다.  
```shell  
Vue.js - The Progressive JavaScript Framework

√ Project name: ... learn-vue3
√ Add TypeScript? ... No / Yes
√ Add JSX Support? ... No / Yes
√ Add Vue Router for Single Page Application development? ... No / Yes
√ Add Pinia for state management? ... No / Yes
√ Add Vitest for Unit Testing? ... No / Yes
√ Add an End-to-End Testing Solution? » No
√ Add ESLint for code quality? » Yes
√ Add Prettier for code formatting? ... No / Yes  
```  
  
선택을 완료 하면 프로젝트가 생성되는데 이후 의존성 추가 후 기동을 하면 동작한다.  
```shell  
cd /{myproject}
npm install  
```  
  
`npm run dev` 를 하게 되면 아래와 같이 로컬에서 접속 할 수 있는 주소가 뜬다.
